# IoT Smart Monitoring System for Refrigeration

## Overview
An ESP32-based IoT system that monitors refrigerator conditions such as door status, gas leaks, and food spoilage, with real-time alerts and cloud logging.

## Features
- Door status detection using LDR
- Gas leak detection using MQ-2 sensor
- Food spoilage detection using MQ-3 sensor
- Real-time alerts via Pushbullet API
- Cloud logging using ThingSpeak

## Tech Stack
ESP32, Arduino IDE, ThingSpeak, Pushbullet API

## System Flow
Sensors → ESP32 Processing → Threshold Logic → Cloud + Alerts
